Introducción¶

El anlisis de regresin es uno de los mtodos estadsticos ms usados para analizar relaciones entre variables, por esta razn estos mtodos son usados muy a menudo para inferir sobre problaciones usando muestras representativas de dichas poblaciones que nos de una idea de lo que ocurre a nivel general. Este mtodo de anlisis tiene la caracterstica de poseer una variable respuesta, la cual es explicada por variables independientes o predictoras que se cree tienen algn tipo de relacin con la variable respuesta. Por lo regular, cuando se hace un estudio se plantea una hiptesis sobre la variable a investigar, luego se procede a observar las variables que se cree (Por parte del investigador)influyen en dicha variable respuesta, planteando asi un modelo de regresin adecuado que nos permita verificar cuales son variables significativas y cuales no lo son, descartando asi aquellas variable que no son relevantes en el proceso.

NOTA: Las variables predictoras deben ser independientes dos a dos.

Historia¶

Las ideas modernas sobre regresin se originan en el trabajo de Legendre y Gauss, sobre el mtodo de mnimos cuadrados, para ajustar los datos sobre las rbitas de cuerpos celestes. El primer estudio documentado sobre el mtodo de mnimos cuadrados, de donde deriva la idea de regresin, es debido a Legendre (1752-1833) en 1805. Esta tcnica de optimizacin intenta encontrar la funcin (dentro de una familia) que mejor se ajusta a los datos bivariantes, de acuerdo con el criterio de mnimo error cuadrtico, y siempre que los datos cumplan algunas condiciones (como independencia). Se conocan las ecuaciones funcionales de estas rbitas, pero los errores de medida haca que los clculos fuesen aproximados y se ajustaban ciertas familias de funciones, usando la teora de errores (y la distribucin normal para describirlos). En 1829 Gauss fue capaz de establecer la razn por la cual este procedimiento es muy adecuado desde el punto de vista estadstico, mediante lo que hoy se conoce como teorema de Gauss-Markov, que muestra que los estimadores obtenidos con este mtodo son insesgados y no se requiere una distribucin especfica para los datos que se ajustan. El impulso posterior lo dan los trabajos citados de Galton y Pearson sobre la herencia. Yule la utiliza para el estudio de fenmenos sociales, como las causas de pobreza alrededor de 1899.

A lo largo del siglo XIX el trabajo de los estadsticos era mayormente descriptivo; la inferencia estadstica se va a desarrollar como consecuencia de la creacin de la Escuela Biomtrica del University College de Londres bajo la direccin del matemtico Karl Pearson (1857-1936), quien trata de aportar bases matemticas a los descubrimientos de Galton (Botasso, 2009). Pearson defiende que el mtodo cientfico es esencialmente estadstico, pues sus inferencias se basan en la asociacin entre antecedentes y consecuentes. Alrededor de 1895, Pearson haba resuelto las propiedades matemticas del coeficiente de correlacin y la regresin simple utilizada para la prediccin lineal entre dos variables continuas. Tambin generaliza la idea de regresin de Galton a varias dimensiones, comprobando que muchas variables biomtricas siguen la distribucin normal multivariante y por tanto se podra aplicar la regresin mltiple para relacionar varias de estas medidas. Yule introduce la notacin e ideas para la correlacin parcial de dos variables fijando una tercera en un artculo Sobre la teora de la correlacin para cualquier nmero de variables tratado mediante una nueva notacin publicado tambin en los Proceedings de la Royal Society en 1907.

Planteamiento del algoritmo¶

$\hspace{0.2cm}$

Planteamiento matemtico

Modelo de regresin

$Y_i=\beta_0 + \beta_{1i} X_{1i}+ \beta_{2i} X_{2i}+...+ \beta_{ki} X_{ki} + \epsilon_i$

Se tiene que $X_{1},X_{2},...,X_{k}$ son la variables independientes del modelo, estas explican a la variable $Y_i$. La variable respuesta $Y_i$ depende de las variables explicativas junto con un componente de error $\epsilon_i$, el cual se distribuye con media cero y varianza constante.

Supuestos del modelo

(i) $E(\epsilon_i)=0$

(ii) $E(\epsilon_i^2)=\sigma^2$

(iii) $E(\epsilon_{i}\epsilon_{i'})=0$

(iv) $\epsilon_i$~ $N(0,\sigma^2)$

Ajuste del modelo

Para ajustar el modelo se realiza el mtodo de mnimos cuadrados o el de mxima verosimilitud. Estos mtodos coinciden en la estimacin de parametros del modelo, por lo cual queda a gusto del lector realizar el de su preferencia.

Una vez se estiman los parmetros del modelo se pueden obtener estimaciones de la variable dependiente $Y_i$. Estas estimaciones tienen la siguiente forma.

$\hat{Y}_i=\hat{\beta}_0 + \hat{\beta}_{1i} X_{1i}+ \hat{\beta}_{2i} X_{2i}+...+ \hat{\beta}_{ki} X_{ki}$

El error de cada estimacin con respecto a la variable respuesta original es:

$\epsilon_i = Y_i - \hat{Y}_i = Y_i - (\hat{\beta}_0 + \hat{\beta}_{1i} X_{1i}+ \hat{\beta}_{2i} X_{2i}+...+ \hat{\beta}_{ki} X_{ki})$

donde $\hat{\beta_0},\hat{\beta_1},...,\hat{\beta_k}$ son los parmetros estimados del modelo. El criterio de mnimos cuadrados asigna a $\hat{\beta_0},\hat{\beta_1},...,\hat{\beta_k}$ valores que minimizan la suma de errores al cuadrado de todas las observaciones.

import numpy as np
import random
from sklearn import linear_model
from sklearn.metrics import mean_squared_error, r2_score
import matplotlib.pyplot as plt
%matplotlib inline
 
# Generador de distribucin de datos para regresin lineal simple
def generador_datos_simple(beta, muestras, desviacion):
  x = np.random.random(muestras) *100
  e = np.random.randn(muestras) * desviacion
  y = x * beta + e
  return x.reshape((muestras,1)), y.reshape((muestras,1))
 
# Parmetros de la distribucin
desviacion = 200
beta = 10
n = 50
x, y = generador_datos_simple(beta, n, desviacion)
 
# Represento los datos generados
plt.scatter(x, y)
plt.show()

Conclusiones¶

El anlisis de regresin es una de las mejores tcnicas para analizar relaciones entre variables. En este documento se expreso el modelo general (multivariado) de regresin lineal con la intencin de ampliar la curiosidad del lector, este modelo puede usarse con variables cualitativas expresadas como variables dummy, por lo cual nos brinda una amplia posibilidad sin discriminar entre variables cualitativas y cuantitativas.

Es importante en este modelo el cumplimiento de los supuestos, esto debido a que la inferencia respecto a los datos que nos da el modelo tiene soporte en la distribucin normal, por lo cual si no se da cumplimiento a los supuestos la inferencia no tiene sentido.

Nota: los datos deben tener un comportamiento normal, en caso de no cumplirse, lo indicado es hacer una transformacin sobre los datos que nos permita obtener normalidad. Si la transformacin no ayuda deberiamos extendernos a modelos lineales generalizados
